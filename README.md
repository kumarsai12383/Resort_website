# Aethelgard Luxury Hotel

A modern, animated landing page for an ultra-luxury resort experience. Built with React and Vite, featuring rich visuals, motion-driven sections, and a clean component structure.

## Features

- Hero, about, amenities, rooms, and offers sections
- Gallery with lightbox experience
- Testimonials, FAQ, and timeline storytelling
- Scroll progress indicator and animated stats
- Responsive layout with Tailwind CSS

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 3
- Framer Motion
- GSAP

## Getting Started

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
